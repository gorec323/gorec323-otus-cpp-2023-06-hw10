Подробное описание

A special type, similar to std::nothrow_t, used to disambiguate constructors that accept executor arguments. The executor_arg_t struct is an empty structure type used as a unique type to disambiguate constructor and function overloading. Specifically, some types have constructors with executor_arg_t as the first argument, immediately followed by an argument of a type that satisfies the Executor type requirements.
