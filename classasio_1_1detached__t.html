Подробное описание

A completion_token type used to specify that an asynchronous operation is detached. The detached_t class is used to indicate that an asynchronous operation is detached. That is, there is no completion handler waiting for the operation's result. A detached_t object may be passed as a handler to an asynchronous operation, typically using the special value asio::detached. For example:

my_socket.async_send(my_buffer, asio::detached);
